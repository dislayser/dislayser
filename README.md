# Портфолио DISLAYSER

## Запуск локально

```bash
npm install
npm run dev -- --host 0.0.0.0 --port 5051
```

Откройте `http://localhost:5051/`.

## Контент

Все данные портфолио меняются в `src/content.js`.

## Docker Compose

```bash
docker compose up --build
```

Откройте `http://localhost:5051/`.
